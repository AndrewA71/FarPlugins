
Плагин тестировался в Far Manager 3.0.

Для работы необходимы:
1. Pygin — Python adapter, оригинальный находится здесь:
   https://github.com/trexinc/evil-programmers/tree/master/pygin.
   Мой исправленный и скомпилированный здесь:
   https://github.com/AndrewA71/Pygin/releases/download/20221210/pygin_20221210.zip
2. Python 3.10 или выше с установленными библиотеками PyYAML и pywin32.
   Путь к Python должен быть добавлен в path.

Устанавливать обычным способом:
 * создайте каталог с произвольным именем в каталоге %FARHOME%\Plugins;
 * скопируйте туда все, что находится в дистрибутиве плагина;
 * если установлен плагин Renewal, то скопируйте в каталог плагина Renewal
   файл Renewal\RenewalCustom.xml (или добавьте к существующему) для
   возможности автоматического обновления на новые версии;
 * перезапустите FAR.
